# 🛡️ Week 3 — Log Parsing and Threat Intelligence Extraction  

## 📌 Week 3 Overview  
Week 3 focuses on transforming raw Cowrie honeypot logs into actionable cybersecurity intelligence.

After successfully deploying and exposing the healthcare IoT honeypot in Weeks 1 and 2, this phase targets data engineering and threat analysis. The objective is to parse large JSON/text logs generated by Cowrie, extract Indicators of Compromise (IoCs), identify attacker behavior patterns, and prepare structured threat intelligence for visualization in Week 4.

---

# 🎯 Week 3 Objectives  

### Primary Goals:
- Parse Cowrie JSON logs using Python  
- Extract attacker IP addresses  
- Identify brute-force attempts  
- Detect successful login attempts  
- Extract executed terminal commands  
- Identify malware download attempts  
- Generate CSV/JSON intelligence reports  
- Prepare structured datasets for dashboarding
